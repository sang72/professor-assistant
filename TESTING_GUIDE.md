## 🎯 주요 기능 설명

### 1. 목차 파일 업로드 방법
**파일 형식 (예시):**
```
Chapter 1: Introduction to Franchising
Chapter 2: Franchise Relationship Model
Chapter 3: Wealth Creating Power
```

또는 마크다운:
```
# Chapter 1: Introduction to Franchising
# Chapter 2: Franchise Relationship Model
# Chapter 3: Wealth Creating Power
```

**업로드 방법:**
1. [교재 관리] 탭 선택
2. 목차 파일 드래그 & 드롭 또는 [선택] 클릭
3. .txt 또는 .md 파일 선택
4. 업로드 완료 메시지 확인

### 2. 챕터별 파일 업로드 방법
1. 목차 파일이 업로드된 후 챕터 목록 표시
2. 각 챕터 옆 [선택] 버튼 클릭
3. PDF, 텍스트, DOC 파일 선택
4. 업로드 완료

### 3. 프롬프트 복사 방법
1. [작업 목록] 탭 선택
2. 각 항목의 "📋 복사" 버튼 클릭
3. 프롬프트가 클립보드에 복사됨 (✅ 복사됨 메시지)
4. Claude Code에 붙여넣기 후 실행

### 4. Word 파일 다운로드 방법
1. [작업 목록] 탭에서 "📄 Word" 버튼 클릭
2. 또는 [파일 다운로드] 탭에서 파일 선택
3. .docx 파일이 자동으로 다운로드됨

## 🐛 문제 해결

### Q: 포트 3000이 이미 사용 중입니다
**A:** 
```bash
# Windows PowerShell에서:
$conn = Get-NetTCPConnection -LocalPort 3000 -ErrorAction SilentlyContinue
Stop-Process -Id $conn.OwningProcess -Force

# 그 후 다시 시작:
./run-web.bat
```

### Q: 목차 파일을 업로드해도 챕터 목록이 나타나지 않습니다
**A:** 
1. 파일 형식 확인 (Chapter N: Title 형식)
2. 페이지 새로고침 (F5)
3. 브라우저 콘솔 확인 (F12 → Console)

### Q: Word 파일이 다운로드되지 않습니다
**A:** 
1. 강의안이 생성되어 있는지 확인
2. 백엔드 서버 로그 확인
3. 브라우저 다운로드 폴더 확인

### Q: 프롬프트 복사가 작동하지 않습니다
**A:** 
1. 최신 브라우저 사용 (Chrome, Firefox, Safari, Edge)
2. HTTPS 또는 localhost에서만 작동
3. 브라우저 권한 확인 (클립보드 접근)
