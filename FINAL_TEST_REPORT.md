### API 엔드포인트 (총 10개)

```
GET /health
Response: {"status":"ok","message":"Professor Assistant Server is running"}

```

#### 2. 교과목 목록 ✅
```
GET /api/courses
Response: 1개 교과목 반환 (Franchise Management)
Status: 200 OK
Time: < 100ms
```

#### 3. 교과목 상세정보 ✅
```
GET /api/courses/:folder
Response: 완전한 과목 정보 + 상태 데이터
Fields: course_name, professor_name, status, progress(22%)
Status: 200 OK
```

#### 4. 목차 조회 ✅
```
GET /api/courses/:folder/toc
Response: 빈 배열 (아직 TOC 업로드 안함)
Status: 200 OK (에러 처리됨)
```

#### 5. 챕터 목록 ✅
```
GET /api/courses/:folder/chapters
Response: {"chapters":[]}
Status: 200 OK
```

#### 6. Word 파일 생성 ✅
```
GET /api/courses/:folder/docx/script-week01-session1
Response: 19KB .docx 파일
File Type: Microsoft Word 2007+
Status: 200 OK
Time: < 1s
```

#### 7. 파일 목록 조회 ✅
```
GET /api/courses/:folder/files
Response: 파일 목록 배열
Status: 200 OK
```

#### 8. 프롬프트 생성 (구현됨)
```
GET /api/courses/:folder/prompt/:task
Status: 200 OK
Response: Claude 프롬프트 문자열
```

#### 9. 목차 파일 업로드 (구현됨)
```
POST /api/courses/:folder/upload-toc
File Types: .txt, .md
Status: 201 Created
```

#### 10. 챕터 파일 업로드 (구현됨)
```
POST /api/courses/:folder/chapters/:key
File Types: .pdf, .txt, .doc, .docx
Status: 201 Created
```
