### Main Content

#### Segment 1: Building Complete Financial Models (15 minutes)

**The Complete Franchisee P&L Model:**

Let's build a full financial model for a chicken franchise in a good Seoul location. I'll break it down into five components:

**1. Revenue Projections**

First, we estimate monthly sales. How do we do this?

- Count daily transactions (observe location during peak hours)
- Estimate average transaction value
- Extrapolate to monthly

Example:
- Location: Bussan subway station, 10-minute walk away
- Observation: During lunch (11am-2pm), ~80 transactions/hour, ~3 hours = 240 transactions
- Evening (5pm-8pm), ~40 transactions/hour, ~3 hours = 120 transactions
- Off-peak (rest of day), ~20 transactions/hour, ~15 hours = 300 transactions
- **Total daily: 660 transactions**
- Average transaction: 12,000 won (mix of single orders and group orders)
- **Daily revenue: 7.92M won**
- **Monthly revenue: 237.6M won (30 days)**

This is high-end estimate (good location).

**2. Variable Costs**

Based on our unit economics framework:

- **COGS:** 30% of sales = 71.28M
- **Direct labor** (cooking, prep, packing): 10% of sales = 23.76M
- **Packaging** (boxes, napkins, side containers): 3% of sales = 7.128M
- **Total Variable Costs:** 43% of sales = 102.14M

**3. Fixed Costs**

Costs that don't change with volume:

- **Rent:** 18% of sales (this location's market rate) = 42.77M
- **Utilities:** 3% of sales = 7.128M
- **Equipment depreciation:** 2% of sales = 4.75M
- **Insurance:** 1% of sales = 2.38M
- **Owner salary:** 30M (living wage for franchisee)
- **Miscellaneous:** 5% of sales = 11.88M
- **Total Fixed Costs:** 98.89M

**4. Franchise Fees**

- **Royalty:** 6% of sales = 14.26M
- **Marketing fund:** 1% of sales = 2.38M
- **Total Franchise Fees:** 16.64M

**5. Complete P&L**

| Item | Amount | % of Sales |
|------|--------|-----------|
| **REVENUE** | **237.6M** | **100%** |
| Variable costs | -102.14M | -43% |
| **Gross Profit** | **135.46M** | **57%** |
| | | |
| **FIXED COSTS** | | |
| Rent | -42.77M | -18% |
| Utilities | -7.128M | -3% |
| Depreciation | -4.75M | -2% |
| Insurance | -2.38M | -1% |
| Owner salary | -30M | -12.6% |
| Miscellaneous | -11.88M | -5% |
| **Subtotal Fixed** | **-98.89M** | **-41.6%** |
| | | |
| **FRANCHISE FEES** | | |
| Royalty | -14.26M | -6% |
| Marketing | -2.38M | -1% |
| **Subtotal Franchisee Fees** | **-16.64M** | **-7%** |
| | | |
| **NET PROFIT** | **19.93M** | **8.4%** |

**Wait, let me recalculate because I need to check if I'm over-allocating owner salary. Let me simplify:**

Actually, owner salary is already implicit in profit. Let me recalculate without explicit owner salary line, which should be paid from profit.

| Item | Amount | % of Sales |
|------|--------|-----------|
| **REVENUE** | **237.6M** | **100%** |
| | | |
| **COGS & Direct Costs** | | |
| Food COGS | -71.28M | -30% |
| Direct labor | -23.76M | -10% |
| Packaging | -7.128M | -3% |
| **Subtotal Variable** | **-102.14M** | **-43%** |
| | | |
| **OPERATING EXPENSES** | | |
| Rent | -42.77M | -18% |
| Utilities | -7.128M | -3% |
| Equipment maintenance | -4.75M | -2% |
| Insurance | -2.38M | -1% |
| General labor (management, admin) | -17.856M | -7.5% |
| Marketing (local) | -2.38M | -1% |
| Miscellaneous | -5.94M | -2.5% |
| **Subtotal Operating** | **-82.85M** | **-34.9%** |
| | | |
| **FRANCHISOR FEES** | | |
| Royalty (6%) | -14.26M | -6% |
| Marketing contribution (1%) | -2.38M | -1% |
| **Subtotal Fees** | **-16.64M** | **-7%** |
| | | |
| **NET PROFIT (EBITDA)** | **36.27M** | **15.3%** |

**Annual profit: 435.24M**
**5-year cumulative profit: 2.176B**

This is a strong franchisee location. Let me calculate financial metrics:

**Key Financial Metrics:**

- **Monthly profit:** 36.27M
- **Annual profit:** 435.24M
- **Initial investment:** 250M (equipment, inventory, deposits)
- **Payback period:** 250M ÷ 36.27M = 6.9 months ✓ EXCELLENT
- **Annual ROI:** 435.24M ÷ 250M = 174% ✓ OUTSTANDING
- **Net profit margin:** 15.3% ✓ HEALTHY

This franchisee is in an exceptional location and will succeed.

#### Segment 2: Scenario Analysis & Sensitivity Testing (15 minutes)

**Now let's stress-test this model. What if things don't go as planned?**

**SCENARIO 1: Sales Miss (20% Below Projection)**

If sales drop to 190M/month instead of 237.6M:

| Item | Original | Scenario 1 | Change |
|------|----------|-----------|--------|
| Revenue | 237.6M | 190M | -20% |
| Variable costs (43%) | -102.14M | -81.7M | -20% (proportional) |
| Fixed costs | -82.85M | -82.85M | 0% (same) |
| Franchise fees (7%) | -16.64M | -13.3M | -20% |
| **NET PROFIT** | **36.27M** | **11.15M** | **-69%** |

Profit collapses by 69%! Even with a 20% sales decline, franchisee is still profitable (11.15M > 0), but the margin drops from 15.3% to 5.9%.

**Payback period:** 250M ÷ 11.15M = 22.4 months (still acceptable, but much slower)

**SCENARIO 2: Rent Increase (From 18% to 22%)**

Korean landlords often renegotiate after 3-5 years. What if rent increases by 4 percentage points?

Original rent: 42.77M (18% of 237.6M)
New rent: 52.1M (22% of 237.6M)
Additional cost: 9.33M

| Item | Original | Scenario 2 | Change |
|------|----------|-----------|--------|
| Rent | -42.77M | -52.1M | +9.33M cost |
| **NET PROFIT** | **36.27M** | **26.94M** | **-26%** |

A 4-percentage-point rent increase cuts profit by 26%. But franchisee is still profitable.

**Why? Because revenue is still 237.6M. The base is large enough to absorb rent increase.**

**SCENARIO 3: Both Sales Miss AND Rent Increase**

This is the real stress test:

- Sales drop to 190M (-20%)
- Rent stays at 22% of sales (landlord doesn't reduce rent)

New rent: 190M × 22% = 41.8M (surprisingly, about same as before, because base sales dropped)

But let me recalculate full P&L:

| Item | Scenario 3 |
|------|-----------|
| Revenue | 190M |
| Variable costs (43%) | -81.7M |
| Fixed costs | -82.85M (mostly same, except rent) |
| Rent | -41.8M (22% of smaller revenue) |
| Other fixed | -41.05M |
| Franchise fees (7%) | -13.3M |
| **NET PROFIT** | **-28.85M** |

**The franchisee is now losing 28.85M/month. This is unsustainable.**

This scenario (sales down 20% + rent negotiates to 22%) transforms a thriving franchisee into a failing one.

#### Segment 3: Case Study - Convenience Store Crisis (13 minutes)

**Real Data: Korean Convenience Store Franchisee, 2026**

A CU franchisee in a busy Seoul subway area. Let's build her model.

**2024 Reality (Good Year):**

| Item | Amount | % of Sales |
|------|--------|-----------|
| Monthly sales | 200M | 100% |
| COGS | -100M | -50% |
| Labor | -50M | -25% |
| Rent | -20M | -10% |
| Utilities | -4M | -2% |
| Equipment maintenance | -2M | -1% |
| Insurance | -2M | -1% |
| Franchisor royalty (3%) | -6M | -3% |
| Misc | -4M | -2% |
| **NET PROFIT** | **12M** | **6%** |

**Annual profit: 144M**
**ROI on 120M investment: 120%**
**Payback: 10 months**

Great numbers. Franchisee is happy.

**2026 Reality (Current Crisis):**

Rent negotiation happened. Landlord demanded increase. COGS increased (supply chain inflation). Labor costs increased (minimum wage).

| Item | Amount | % of Sales |
|------|--------|-----------|
| Monthly sales | 200M | 100% |
| COGS | -110M | -55% |
| Labor | -56M | -28% |
| Rent | -32M | -16% |
| Utilities | -4M | -2% |
| Equipment maintenance | -2M | -1% |
| Insurance | -2M | -1% |
| Franchisor royalty (3%) | -6M | -3% |
| Misc | -4M | -2% |
| **NET PROFIT** | **-16M** | **-8%** |

**The franchisee is LOSING 16M/month.**

**What changed?**
- COGS: 50% → 55% (+5 percentage points)
- Labor: 25% → 28% (+3 percentage points)
- Rent: 10% → 16% (+6 percentage points)
- Total cost increase: 14 percentage points
- Revenue stayed at 200M (sales are stable)
- But profit swung from +12M to -16M (28M loss of value)

**Root cause analysis:**

1. **Rent inflation:** Landlord increased rent by 12M/month (from 20M to 32M)
   - Why? Subway location became more valuable; competing bidders
   - Landlord figured convenience store could absorb increase
   - Franchisor (CU) didn't help negotiate

2. **COGS inflation:** Increased 10M/month
   - Supply chain cost increases across food industry
   - CU tried to absorb some but passed through to franchisees

3. **Labor inflation:** Increased 6M/month
   - Minimum wage mandates
   - Competitive wage pressure (convenience stores need staff)

**Total cost increase:** 28M/month with no revenue increase.

**The Franchisee's Position:**

She has 120M invested and is losing 16M/month. At this rate:
- Annual loss: 192M
- She'll lose her entire investment in ~8 months of operation
- She has no cushion

**Options:**

1. **Renegotiate with landlord**
   - Difficult (she's locked into lease)
   - Might get 1-2M rent reduction
   - Would help but not solve the problem

2. **Improve efficiency**
   - Reduce labor (hire cheaper staff, fewer hours)
   - But service suffers; customers go elsewhere
   - Could save 2-3M but risk sales decline

3. **Raise prices**
   - Impossible for convenience store (prices are competitive)
   - 10% price increase = 20M more revenue, but customers switch to competitor

4. **Negotiate with franchisor**
   - Ask for royalty reduction
   - CU sometimes does this in crisis zones
   - Could get royalty reduced from 3% to 2%
   - Saves: 2M/month (not enough)

5. **Exit the franchise**
   - Sell location to another franchisee (buyer knows profits are negative—low price)
   - Close and try to recover deposit
   - Declare bankruptcy

**This franchisee is likely heading to bankruptcy. The system failed her.**

**Why Did CU's System Fail Here?**

CU's system is excellent when:
- Rent is stable
- COGS is manageable
- Labor market is competitive

But when:
- Rent inflation accelerates (Seoul's problem since 2020)
- Supply chain costs rise (post-COVID, 2021-2025)
- Minimum wage increases (government policy, 2015-2025)

The system breaks because all these costs hit simultaneously, and there's no shock absorber.

**If CU Had Been Smarter:**

CU could have:
1. Signed longer leases at fixed rates (before prices rose)
2. Invested in supply chain to reduce COGS (but this is costly)
3. Diversified revenue (expanded delivery, prepared foods, services)
4. Shared pain during crisis (reduce royalty when franchisee profit drops below 5%)

But CU didn't do these things. They stuck to the model even as the market changed.

**Result:** Hundreds of CU franchisees are struggling. The golden system is tarnished.

### Summary (2 minutes)

**Key Learnings:**

1. **Financial models** help predict franchisee success before signing
2. **Scenario analysis** reveals how sensitive profit is to changes
3. **Even excellent franchises fail** if macro conditions shift negatively
4. **Rent inflation** is the #1 killer in Korea (more than COGS or labor)
5. **System resilience** depends on franchisor willingness to share pain during downturns

## SECTION 4: KEY NOTES & IMPROVISATION TIPS

### Instructor Preparation Notes

**Before the Session:**
- Have a working spreadsheet with formulas
- Prepare 2-3 real franchise cases with actual numbers
- Optional: Video of struggling franchisee talking about rent crisis
- Have current CU/GS25/Lotte financial data (if public)

**During the Session:**
- Build the model live; don't just show slides
- Show students how to change numbers and watch profit change
- Ask: "What would need to change for this to be profitable?" after scenario 3
- Emphasize: Even small percentage-point changes have massive profit impact

### Vocabulary Emphasis

- **Financial model:** Spreadsheet showing revenue, costs, profit
- **Scenario analysis:** Testing "what if" situations
- **Sensitivity:** How much profit changes from small cost/volume changes
- **Break-even scenario:** Sales level where profit = zero
- **Fixed-cost ratio:** Fixed costs as % of revenue
- **Macro shock:** Economic change (inflation, rent increase) affecting all franchisees
- **System resilience:** How well system survives external shocks
