### Frequently Asked Questions

**Q1: "How do I approach a complex case on the exam?"**

**A:** Use this 5-step process:

**Step 1: Problem Identification (2 min)**
- Read case once, identify core problem
- "What's actually wrong here?"
- Example: "Sales declining, franchisees struggling, high failure rate"

**Step 2: Framework Selection (2 min)**
- Which concepts apply?
- Four mechanisms? Six red flags? SDS issues? Financial problem?
- Example: "Mechanisms 2 and 3 failing; Red flags 2, 3, and 4 present"

**Step 3: Detailed Analysis (8 min)**
- Apply framework to specific case details
- Provide evidence from case
- Example: "Red Flag #2 (oversaturation): Case mentions 25 competing franchises in same area. Franchisee sales declining from 30M to 20M confirms oversaturation."

**Step 4: Root Cause Diagnosis (2 min)**
- Why did problem occur?
- What did franchisor/franchisee do wrong?
- Example: "Franchisor prioritized rapid expansion over location quality. Weak SDS means no competitive differentiation."

**Step 5: Recommendation (1 min)**
- What should company do NOW?
- What should company do LONG-TERM?
- Example: "SHORT-TERM: Stop new openings. Address profitability crisis immediately. LONG-TERM: Build stronger SDS and be more selective about locations."

**Q2: "The four mechanisms seem to appear in every answer. Do I need to mention them always?"**

**A:** Good question. Here's the rule:

- **Multiple choice:** Sometimes directly relevant, sometimes not. Read question carefully.
- **Short answer:** Usually relevant. Reference mechanisms if applicable.
- **Case analysis:** ALWAYS use as framework. Explicitly state which mechanisms are working/failing.

The mechanisms are like a diagnostic tool. Use it when analyzing franchise viability.

**Q3: "How do I distinguish between Location Problem vs. Franchisee Problem?"**

**A:** This distinction is crucial because the blame differs:

**Location Problem:**
- Evidence: Sales declining even though franchisee following system
- Cause: Area demographics changed, competitor opened, oversaturation
- Fault: Franchisor (should have chosen better location)
- Solution: Relocate or renegotiate economics

**Franchisee Problem:**
- Evidence: Sales declining because franchisee not following SDS, poor management
- Cause: Franchisee incompetence or non-compliance
- Fault: Franchisee
- Solution: Training, monitoring, replacement if needed

**How to Tell (on Exam):**
- Read case: Does it mention other franchisees in area struggling? → Location problem
- Does it mention this franchisee not following procedures? → Franchisee problem
- Are all franchisees profitable? → Good locations and good franchisee
- Are all franchisees unprofitable? → Location or system problem

**Q4: "How much detail should I provide in answers?"**

**A:** Balance depth and brevity:

- **Too much:** Writing everything you know (wastes time, unclear thinking)
- **Too little:** One sentence with no explanation (insufficient depth)
- **Right amount:** 
  - Multiple choice: Choose one answer with confidence
  - Short answer: 1-2 paragraphs per question (explain using concepts)
  - Case analysis: 4-5 paragraphs (problem → framework → analysis → diagnosis → recommendation)

**Q5: "What if I disagree with the 'correct' answer to a question?"**

**A:** Good critical thinking! Here's how to handle:

- **Multiple choice:** Pick best answer even if imperfect
- **Short answer:** Provide your answer with clear reasoning. Partial credit awarded if reasoning is sound.
- **Case analysis:** If you interpret case differently, explain your interpretation. Rubric values sound reasoning over "correct" answer.

**Example:** If case says "franchisee earned 50M won," but you argue "that's impossible given stated expenses," explain your math. You may get credit for critical thinking.

**Q6: "Should I use Korean examples or international examples?"**

**A:** Both are valuable:

- **Korean examples:** Show you understand local context (major advantage)
- **International examples:** Show broader understanding (also good)
- **Best:** Mix both (shows comprehensive knowledge)

Exam values original thinking, so use examples that support YOUR analysis.

**Q8: "What if I don't know something?"**

**A:** Three strategies:

1. **Skip and return:** If stuck on question, skip and return if time permits
2. **Partial answer:** Write what you DO know. Partial credit awarded.
3. **Admit uncertainty:** "I'm not certain, but based on [concept], my answer is..." Shows reasoning even if uncertain.
